# Ink & Strategy

Blog de veille sur le tatouage et la communication digitale pour les studios de tattoo.

## À propos

Ce blog explore l'univers du tatouage sous toutes ses facettes : apprentissage des techniques, hygiène, parcours d'artistes inspirants et stratégies de communication digitale.

## Stack technique

- **SSG** : Hugo
- **Thème** : Pehtheme Hugo
- **Styling** : TailwindCSS
- **Hébergement** : GitHub Pages
- **Contenu** : Markdown

## Lancer le projet en local

```bash
hugo server
```

Le site sera accessible sur `http://localhost:1313`
